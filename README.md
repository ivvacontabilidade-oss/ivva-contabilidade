# IVVA Contabilidade - Site Profissional

Este é o código-fonte do site profissional da IVVA Contabilidade, desenvolvido com tecnologias modernas para garantir alta performance e segurança.

## Tecnologias Utilizadas
- **React:** Biblioteca para interfaces de usuário (usada por Facebook, Netflix, etc).
- **Tailwind CSS:** Framework de estilos para design moderno e responsivo.
- **Vite:** Ferramenta de construção ultra-rápida.
- **Lucide React:** Ícones leves e bonitos.

## Como Editar o Conteúdo
Os textos e informações do site estão nos arquivos dentro da pasta `src/`:
- `src/components/Header.tsx`: Cabeçalho e Menu.
- `src/components/Footer.tsx`: Rodapé e informações de contato.
- `src/pages/Home.tsx`: Página inicial.
- `src/pages/About.tsx`: Página Sobre Nós.
- `src/pages/Services.tsx`: Página de Serviços.
- `src/pages/Contact.tsx`: Página de Contato.

## Como Publicar
Consulte o arquivo `MELHOR_OPCAO_PUBLICACAO.md` para um guia detalhado sobre as melhores opções de hospedagem.

## Comandos Úteis
- `npm run dev`: Inicia o servidor de desenvolvimento local.
- `npm run build`: Gera a versão final otimizada para publicação (cria a pasta `dist`).
- `npm run preview`: Visualiza a versão final localmente.
